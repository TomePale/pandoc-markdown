# Korisni resursi

Ovaj dokument sadrži popis korisnih linkova i resursa za učenje Markdowna i Pandoca.

## Službena dokumentacija

### Markdown

- [Markdown - Službena stranica](https://daringfireball.net/projects/markdown/) - Originalna Markdown dokumentacija by John Gruber
- [GitHub Flavored Markdown](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) - GitHub proširenja Markdown sintakse
- [CommonMark](https://commonmark.org/) - Standardizirana specifikacija Markdowna
- [Markdown Guide](https://www.markdownguide.org/) - Sveobuhvatni vodič za Markdown

### Pandoc

- [Pandoc - Službena stranica](https://pandoc.org/) - Glavni izvor dokumentacije


## Interaktivni alati

- [HackMD](https://hackmd.io/) - Online Markdown editor s live preview
- [StackEdit](https://stackedit.io/) - Markdown editor u pregledniku s sync mogućnostima
- [Markdown Live Preview](https://markdownlivepreview.com/) - Brz online pregled Markdown sintakse
- [Pandoc Playground](https://pandoc.org/try/) - Isprobajte Pandoc konverzije online

## Tutoriali i vodiči

- [Markdown Tutorial](https://www.markdowntutorial.com/) - Interaktivni tutorial za početnike
- [Learn Markdown in Y Minutes](https://learnxinyminutes.com/docs/markdown/) - Brzi pregled sintakse
- [Pandoc for beginners](https://programminghistorian.org/en/lessons/sustainable-authorship-in-plain-text-using-pandoc-and-markdown) - Detaljni vodič za početnike

## Alati za konverziju

- [Pandoc](https://pandoc.org/) - Univerzalni konvertor dokumenata
- [Kramdown](https://kramdown.gettalong.org/) - Ruby Markdown parser s naprednim funkcijama
- [Marked](https://marked.js.org/) - JavaScript Markdown parser
- [markdown-it](https://github.com/markdown-it/markdown-it) - Markdown parser u JavaScriptu

## Editori s Markdown podrškom

- [Visual Studio Code](https://code.visualstudio.com/) - Sa ugrađenim Markdown preview
- [Typora](https://typora.io/) - WYSIWYG Markdown editor
- [Obsidian](https://obsidian.md/) - Alat za bilješke s Markdown podrškom
- [Notable](https://notable.app/) - Markdown editor za bilješke
- [Zettlr](https://www.zettlr.com/) - Markdown editor za akademsko pisanje

- [Markdown Crash Course](https://www.youtube.com/results?search_query=markdown+tutorial) - Brzi uvod u Markdown
